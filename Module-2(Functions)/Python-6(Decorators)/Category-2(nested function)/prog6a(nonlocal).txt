nonlocal  keyword
--------------------
1) def  f1():
	 x = 10
         def  f2():
	         nonlocal  x
	         x = 20
    What  does  x = 20  do  ?  --->  Modifies  outer   function  variable  to  20

2) What  happens  without  nonlocal  x  in  inner  function  f2() ?  --->   x = 20  creates  a  new  LV  with  value  20

3) What  does  nonlocal  x  do ?  ---> Informs  inner  function  to  use  variable  of  outer  function

4) In  other  words,  nonlocal  prevents  inner  function  to  create  (or)  have  a  Lv

5) def  f1():
          def   f2():
               nonlocal  x
    Is  nonlocal  x  valid ?  --->  No  becoz  outer  function  does  not  have  variable  'x'  

6) def  f1():
             nonlocal   x
    Is  nonlocal  x  valid ?  ---> No  becoz  nonlocal  keyword  can  be  used  only  in  inner  function

7) Finally,  who  can  use  nonlocal  word ?  ---> Inner  function  but  outer  function  should  have  the  variable  with  same  name  

8) def  f1():
          x = 10
         def  f2():
               x = 20
               nonlocal  x
    Is  nonlocal  x  valid ?  --->  No  becoz  inner  function  has  already  Lv  'x'  

9) In  other  words,  inner  funtion  can  not  have  a  local  variable  with  same  name  in  order  to  use  nonlocal  keyword
