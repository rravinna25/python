Positional  only  arguments
------------------------------
1) def  f1(a ,  b , /): 
             pass
    What  can  be  passed  to  function  f1() ?  --->   
						Only  positional  arguments  becoz  'a'  and  'b'  are  before  /  in  the  function  header

2) Is  f1(a = 10 , b = 20)  valid ? --->  No  becoz  a = 10  and  b = 20  are  not  positional  arguments
    Is  f1(10 , 20)  valid ?  --->  Yes  becoz  10  and  20  are  positional  arguments

3) In  other  words,  '/'  prevents  in  passing  keyword  arguments  to  the  function

4) def  f2(a , b): 
             pass
    What  can  be  passed  to  function  f2() ?  --->											
						Either  PA's  (or)  KA's  becoz  function  header  does  not  have  *  nor  /

5) def  f3(a , / , b):
                pass
    Which  argument  can  be  passed  for  'a' ?  --->  Only  PA  becoz  it  is  before  /
    Which  argument  can  be  passed  for  'b' ?  --->  Either  PA  (or)  KA  becoz  it  is  not  before  /

6) In  other  words,  arguments  preceding  /   should  be  PA's  only  but
     arguments  following  /  can  be  either  PA's   (or)  KA's

7) def  f4(/ , a , b):
		pass
    Is  the  function  valid ?  --->  No  becoz  arguments  are  missing  before  /

8) In  other  words,  there  should  be  at  least  one  argument  before  /  in  the  function  header



Note:
/  is   permitted  in  the  function   header  from  python  3.1
