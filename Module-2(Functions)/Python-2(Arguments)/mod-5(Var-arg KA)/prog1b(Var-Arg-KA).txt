Variable  number  of  keyword  arguments  (or)  Var-Arg  function
----------------------------------------------------------------------------
1) What  is  variable  number  of  keyword  arguments ?  --->  
										Any  number  of  keyword  arguments  can  be  passed  to  the  function  
								                Eg: f1(Arg1 = Value1 , Arg2 = Value2 , Arg3 = Value3 , .......)

2) def  f1(**a):
	   pass
    Is  f1()  a  var-arg  function ?  --->  Yes  becoz  function  header  has  **
   
3) What  does  **   indicate ?  --->  Var-arg  parameter

4) What  does  pvm  do  when  var-arg  function  is  called ?  --->  														
						Stores  all  the  keyword  arguments  in  a  dictionary  which  is  passed  to  the  function
									
5) f1(EmpNo = 25 , EmpName = 'Sita' , Salary = 10000.0)
    What  is  passed  to  f1()  function  ? ---> A  dictionary  of  three  key : value  pairs

6) Why  are  keyword  arguments  not  passed  to  the  function ?  --->  Since  function  header  has  **

7) f1(Arg1 : Value1 , Arg2 : Value2 , Arg3 : Value3 , .......)
    Is  the  function  call  valid ?  --->  No  becoz  delimeter  should  be  '='  between   the  argument  and  value 

8) f1(No-args)  
    What  is  passed  to  f1()  function ?   ---> Empty dictionary

9) def  f1(**a)
	    pass
     Is  f1(10 , 20 , 30)  valid ?  ---> No  becoz  positional  args  can  not  be  passed  to  the  function  as  function  header  has  **
      
10) In  other  words ,  var-arg  parameter  should  be  keyword  argument  only  due  to  **

11) def  f1(**a = {}):
            pass
     Is  the  function  valid ?  ---> No  becoz  var-arg  parameter  can  not  have  default  value

12) def  f1(x = 25 , **a):
            pass
      Is  the  function  valid ?  --->  Yes  becoz  default  args  and  non-default  args  can  be  in  any  order 
						      due  to  **  in  the  function  header
