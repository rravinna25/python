Types  of  arguments
------------------------
1) Positional  arguments

2) Keyword  arguments

3) Default  arguments

4) Variable  length  positional   arguments

5) Variable  length  keyword  arguments


1) What  is  an  argument ?  --->  The  object  which  is  being  passed  to  the  function  (or)  method  as  input

2) sqrt(x)
    What  is  input  to  sqrt()  function ?  ---> x
    
3) pow(x , y)
    What  are  inputs  to  pow()  function ?  ---> x , y
    
4) What  is  another  name  of  argument ?  ---> Parameter
