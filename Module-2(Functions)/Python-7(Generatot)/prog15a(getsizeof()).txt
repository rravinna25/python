getsizeof()  function
------------------------
1) What  does  getsizeof(object)  do ?  ---> Returns  size  of  the  object  in  terms  of  bytes

2) What  is  the  argument  of  getsizeof()  function ? --->  Any  python  object
											      i.e.   Sequence , non-sequence  and  generator

3) Where  is  getsizeof()  function  defined ?  ---> In  sys  module
