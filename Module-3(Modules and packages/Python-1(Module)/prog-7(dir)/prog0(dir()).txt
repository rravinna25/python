dir()  function
-----------------
1) What  does  dir(module)  do  ?  --->  Returns  all  the  members  of  module  and  environment  variables  in  the  form  of 
							     list  of  strings

2) What  are  the  different  environment  variables ?  --->  __name__ ,  __annotations__ , __builtins__ , __cached__ , 
											      __doc__ ,  __file__ ,  __loader__ ,  __package__ ,  __spec__

3) abc.py
     ------
     import  cal
     x = 25
     def  f1():
     	   pass
     class  c1:
          pass
     What  are  the  members  of  abc  module ?  --->  Object  'x' ,  function  f1() ,  class  c1  and  imported  module  cal

4) In  general,  what  are  members  of  a  module  (Total : 4) ?  --->  Functions , classes , objects  and  imported  modules

5) What  does  dir(No-args)  do ?  --->  Returns  all  the  members  of  current  module  and  environment  variables  in  the 
							      form  of  list  of  strings

6) How  many  arguments  can  dir()  function  take ?  ---> 1  (or)  0

7) dir(mod1)
    Can  mod1  be  used  without  import ?  ---> No    

8) In  other  words,  import  mod1  before  using  mod1

9) Where  is  dir()  function  defined ?  --->  In  builtins  module

10) Can  dir()  function  be  used  without  import ?  --->  
								Yes  becoz  it  is  automatically  imported  as  it  is  a  member  of  builtins  module
